# Biblioteca HTTP API v1

## Authentication

All `/mcp/*` gateway routes and existing dashboard routes are protected by
Cloudflare Access. Requests must arrive with `Cf-Access-Jwt-Assertion`; the
service verifies the JWT against:

- `CF_ACCESS_TEAM_DOMAIN`
- `CF_ACCESS_AUD_TAG`

The authenticated principal is taken from
`Cf-Access-Authenticated-User-Email` for human dashboard users or
`Cf-Access-Service-Token-Name` for machine clients.

Missing, malformed, expired, or wrong-audience JWTs return `401`. Missing
Cloudflare Access configuration returns `503`.

## Gateway Routes

`GET /mcp/health`

Returns registered source names and any source subprocesses currently marked
unavailable.

```json
{
  "sources": ["literature", "lab"],
  "unavailable": {}
}
```

`GET /mcp/sources`

Returns the source registry without subprocess commands or environment values.

```json
{
  "sources": [
    {"name": "literature", "capabilities": ["tool:get_chunks"]},
    {"name": "lab", "capabilities": ["tag:sop"]}
  ]
}
```

`POST /mcp/literature/{tool_name}`

`POST /mcp/lab/{tool_name}`

Forwards the JSON request body as tool parameters to the selected source server.
The successful response is wrapped as:

```json
{
  "result": {}
}
```

If the source is unavailable, the gateway returns `200` with a degraded payload:

```json
{
  "degraded": true,
  "missing_source": "lab",
  "result": null
}
```

Wiki write lock timeouts return `503`.

## Federated Query

`POST /mcp/query`

Request:

```json
{
  "query": "Summarize the latest LFP degradation evidence",
  "mode": "auto",
  "quality": false
}
```

Response:

```json
{
  "answer": "...",
  "intent": "reporting",
  "confidence": 0.9,
  "route": "hybrid",
  "citations": [],
  "sources_consulted": ["literature", "lab"],
  "sources_unavailable": []
}
```

Allowed query modes are `wiki`, `vector`, `graph`, `hybrid`, and `auto`.
